Brain/Circulation Model Developer
BCMD v0.6b

Please read the manual, which can be found in the 'doc' directory.

Quick Start:

  ./configure
  make

Required:

- Python (2.7.x)
- C & Fortran compilers
- Make
- A usable shell, eg. bash

Recommended:

- Tkinter, NumPy, SciPy, Matplotlib
- OpenOpt, PSwarm
- GraphViz
- LaTeX, R
